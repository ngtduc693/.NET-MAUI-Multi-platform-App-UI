# .NET MAUI (Multi-platform App UI)
My repos that use .NET MAUI to make apps run on multi-platform, it is my MAUI template to develop my project that I received from clients

1. .NET version: 8.0
2. Supported OS Platform version:
    - IOS: 16.1+
    - Windows: 10.0+
    - Android: 14.0+
    - Tizen: 6.5+
